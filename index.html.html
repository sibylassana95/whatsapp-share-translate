Installation
Étape par étape

Comment installer l'extension

⚠️

Note : Cette extension n'est pas sur le Chrome Web Store. Elle s'installe en mode développeur, ce qui est simple et sans risque — c'est la méthode standard pour tester des extensions personnelles.

1

Télécharger et décompresser le fichier

Téléchargez le fichier whatsapp-share-translate.zip ci-dessus, puis décompressez-le dans un dossier permanent sur votre ordinateur (ex. : Documents/extensions/whatsapp-share). Ne supprimez pas ce dossier après l'installation.

2

Ouvrir la page des extensions Chrome

Dans Chrome, collez cette adresse dans la barre de navigation et appuyez sur Entrée :

URL Chrome 
chrome://extensions
3

Activer le mode développeur

En haut à droite de la page des extensions, activez le toggle "Mode développeur". De nouveaux boutons apparaissent en haut à gauche.

4

Charger l'extension

Cliquez sur "Charger l'extension non empaquetée", puis naviguez jusqu'au dossier whatsapp-share-translate que vous avez décompressé (le dossier qui contient le fichier manifest.json). Cliquez sur Sélectionner.

5

Épingler l'extension à la barre

Cliquez sur l'icône puzzle 🧩 en haut à droite de Chrome, trouvez "WhatsApp Share + Translate" et cliquez sur l'épingle 📌 pour l'afficher dans la barre d'outils.

6

✅ C'est prêt !

L'icône 📱 apparaît dans votre barre Chrome. Naviguez sur n'importe quelle page (X.com, article de presse, etc.) et cliquez dessus pour partager et traduire.

Utilisation
Guide d'utilisation

Comment utiliser l'extension

A

Méthode 1 — Via le popup (bouton dans la barre)

Sur n'importe quelle page, cliquez sur l'icône 📱 dans la barre Chrome. Le popup affiche un aperçu du contenu traduit. Choisissez votre langue cible, vérifiez la traduction, puis cliquez sur "Partager sur WhatsApp". WhatsApp Web s'ouvre avec le message pré-rempli.

B

Méthode 2 — Via le clic droit sur une sélection

Sélectionnez un passage de texte sur la page (ex. : un tweet, un paragraphe d'article), faites un clic droit, puis choisissez "📱 Partager sur WhatsApp (traduit en français)". Seul le texte sélectionné + le lien sera envoyé.

C

Choisir une autre langue

Dans le popup, utilisez le menu déroulant "Traduire en" pour choisir parmi les 9 langues disponibles. La traduction se met à jour automatiquement.

💡

Pour X.com (Twitter), l'extension détecte automatiquement le texte du tweet. Inutile de le sélectionner manuellement — ouvrez simplement un tweet et cliquez sur l'icône.

Maintenance

Mettre à jour l'extension

→

Remplacer les fichiers

Téléchargez la nouvelle version, remplacez les fichiers dans le dossier original (sans changer son emplacement).

→

Recharger dans Chrome

Allez sur chrome://extensions, trouvez l'extension et cliquez sur l'icône 🔄 (recharger). Les changements sont appliqués immédiatement.
