Description longue
Share & Translate vous permet de partager instantanément n'importe quelle page web vers WhatsApp, LinkedIn ou X (Twitter), avec traduction automatique du contenu dans la langue de votre choix. ✦ FONCTIONNEMENT EN 2 CLICS Ouvrez n'importe quelle page, cliquez sur l'icône Share & Translate dans votre barre Chrome, vérifiez l'aperçu traduit, puis choisissez votre plateforme de partage. ✦ TRADUCTION AUTOMATIQUE Le contenu est traduit automatiquement via Google Translate. La langue source est détectée automatiquement. 9 langues cibles disponibles : français, anglais, espagnol, allemand, italien, portugais, arabe, chinois, japonais. ✦ 3 PLATEFORMES SUPPORTÉES • WhatsApp — app desktop (Windows / macOS) ou WhatsApp Web • LinkedIn — ouvre la boîte de partage avec le texte pré-rempli • X (Twitter) — pré-remplit le tweet avec le contenu traduit ✦ DÉTECTION INTELLIGENTE • Sur X.com : extrait automatiquement le texte du tweet affiché • Sur LinkedIn : extrait le texte d'un post • Sur toute autre page : utilise le titre et la description ✦ PARTAGE PAR SÉLECTION Sélectionnez un extrait de texte sur la page, faites un clic droit, et choisissez la plateforme — seul ce passage est traduit et partagé. ✦ LIENS PRÉSERVÉS Les URLs contenues dans le texte ne sont pas traduites — elles restent intactes dans le message final. ✦ SANS COMPTE NI CLÉ API Aucune inscription requise. Aucune clé API. Totalement gratuit. ✦ CONFIDENTIALITÉ L'extension n'envoie aucune donnée à des serveurs tiers (hors Google Translate pour la traduction). Aucune donnée personnelle n'est collectée ou stockée. Idéal pour les professionnels qui suivent l'actualité internationale, les community managers, les équipes multilingues et toute personne souhaitant partager du contenu traduit rapidement.

Justification des permissions

Google exige une explication pour chaque permission. Voici les textes à renseigner dans le formulaire de soumission :

activeTab
Required to read the URL and title of the currently active tab in order to prepare the share message.
scripting
Required to inject a content script that extracts the page title, meta description, selected text, and tweet content from the active tab.
contextMenus
Required to add a right-click context menu entry that allows users to quickly share selected text to WhatsApp, LinkedIn, or X without opening the popup.
storage
Required to persist the user's preferred target language and WhatsApp opening mode (desktop app or web) across browser sessions.

Privacy Policy — à héberger en ligne
Privacy Policy — Share & Translate Chrome Extension Last updated: 2025 Share & Translate is a Chrome extension that helps users share and translate web page content to WhatsApp, LinkedIn, and X. DATA COLLECTED This extension does not collect, store, or transmit any personal data to its own servers. TRANSLATION Text content is sent to Google Translate API (translate.googleapis.com) solely for the purpose of translation. This is a client-side request made directly from your browser. Please refer to Google's Privacy Policy for details on how Google processes translation requests. LOCAL STORAGE The extension stores two user preferences locally in your browser using Chrome's storage API: - Your preferred target language - Your preferred WhatsApp opening mode (app or web) This data never leaves your device. PERMISSIONS - activeTab: To read the current page URL and title - scripting: To extract page content (title, description, selected text) - contextMenus: To add a right-click share option - storage: To save your preferences locally THIRD-PARTY SERVICES The only third-party service used is Google Translate (translate.googleapis.com). No data is shared with WhatsApp, LinkedIn, or X beyond what you explicitly choose to share through their standard sharing interfaces. CONTACT For questions about this privacy policy, please open an issue on the extension's GitHub repository.
Soumission
Étape par étape

Créer un compte & soumettre

Frais uniques : L'inscription au programme développeur Google coûte 5 $ USD (une seule fois, à vie). Prévoie une carte bancaire.

1

Créer un compte développeur Google

Va sur chrome.google.com/webstore/devconsole, connecte-toi avec ton compte Google et accepte les conditions. Paie les 5 $ par carte bancaire. Le compte est activé immédiatement.

2

Héberger la politique de confidentialité

Crée une page publique avec le texte de confidentialité ci-dessus. Option la plus simple : crée un dépôt GitHub public, ajoute un fichier privacy-policy.md et active GitHub Pages. Tu obtiendras une URL du type https://tonpseudo.github.io/share-translate/privacy-policy.

3

Créer le fichier ZIP à soumettre

Décompresse share-translate-v3.zip. Dans le dossier, vérifie que manifest.json est présent. Crée un nouveau ZIP du contenu du dossier (pas du dossier lui-même). Ne pas inclure les fichiers cachés (.DS_Store, .git).

4

Créer une nouvelle extension dans la console

Dans la Developer Console, clique sur "Add new item", puis "Browse files" et upload le ZIP. La console valide le format automatiquement et affiche les permissions détectées.

5

Remplir la fiche store

Copie-colle les textes de la section "Fiche store" ci-dessus. Ajoute au moins 2 captures d'écran (min. 1280×800 px) du popup en action. La capture "Store icon" doit être 128×128 px (fournie dans les icônes).

6

Renseigner la politique de confidentialité

Dans l'onglet Privacy de la fiche, colle l'URL de la page de confidentialité hébergée. Réponds aux questions sur la collecte de données : non pour toutes sauf "Does your extension use remote code?" → non.

7

Justifier les permissions

Dans l'onglet Permissions, colle les justifications de la section "Justification des permissions" ci-dessus pour chacune des 4 permissions.

Soumettre pour révision

Clique sur "Submit for review". Google examine l'extension dans un délai de 1 à 3 jours ouvrés. Tu recevras un email de confirmation. En cas de refus, Google explique la raison et tu peux corriger et resoumettre.

Pour les captures d'écran, tu peux utiliser l'outil de capture Chrome (F12 → Capture node screenshot sur le popup) ou un outil comme Figma pour créer des mockups professionnels. Au moins 2 captures sont requises.
