Share & Translate

Extension Chrome pour partager n'importe quelle page web vers WhatsApp, LinkedIn ou X, avec traduction automatique.

Gratuit Chrome 88+ Manifest V3 Sans clé API v3.0
Nouveautés

Historique des versions

Fonctionnalités
Ce que fait l'extension

Fonctionnalités

Traduction automatique

Détecte la langue et traduit vers le français ou 8 autres langues via Google Translate, sans clé API.

URLs préservées

Les liens dans le texte ne sont pas traduits — ils restent sous leur forme originale dans le message.

Support X.com

Détecte automatiquement le texte des tweets pour une traduction précise sans sélection manuelle.

Support LinkedIn

Extrait le texte des posts LinkedIn et partage via la boîte officielle avec le contenu traduit pré-rempli.

Partage de sélection

Sélectionnez un extrait, clic droit — seul ce passage est traduit et partagé.

Aperçu avant envoi

Le popup affiche le texte traduit avant de l'envoyer, avec possibilité de changer la langue.

App ou Web

WhatsApp : choisissez entre App Desktop et Web. LinkedIn et X ouvrent leur interface native de partage.

Installation
Étape par étape

Comment installer l'extension

Note : Cette extension n'est pas sur le Chrome Web Store. Elle s'installe en mode développeur — méthode standard pour les extensions personnelles, simple et sans risque.

1

Télécharger et décompresser

Téléchargez share-translate-v3.zip et décompressez-le dans un dossier permanent (ex. : Documents/extensions/share-translate). Ne supprimez pas ce dossier — Chrome en a besoin pour fonctionner.

2

Ouvrir la page des extensions Chrome

Collez cette adresse dans la barre Chrome et appuyez sur Entrée :

URL Chrome 
chrome://extensions
3

Activer le mode développeur

En haut à droite de la page des extensions, activez le toggle "Mode développeur". Trois nouveaux boutons apparaissent en haut à gauche.

4

Charger l'extension

Cliquez sur "Charger l'extension non empaquetée", naviguez jusqu'au dossier share-translate-v3 (celui qui contient manifest.json) et cliquez sur Sélectionner.

5

Épingler à la barre Chrome

Cliquez sur l'icône puzzle en haut à droite de Chrome, trouvez "Share & Translate" et cliquez sur l'épingle pour l'afficher en permanence dans la barre d'outils.

Prêt !

L'icône WhatsApp apparaît dans votre barre Chrome. Naviguez sur n'importe quelle page et cliquez dessus pour partager et traduire.

Utilisation
Guide d'utilisation

Comment utiliser l'extension

A

Via le popup — bouton dans la barre

Sur n'importe quelle page, cliquez sur l'icône dans la barre Chrome. Le popup affiche un aperçu du contenu traduit. Choisissez la langue, vérifiez, puis cliquez sur le bouton de la plateforme souhaitée : WhatsApp, LinkedIn ou X.

B

Via le clic droit sur une sélection

Sélectionnez un passage de texte sur la page, faites un clic droit, puis choisissez "Share & Translate" et la plateforme cible dans le sous-menu. Seul le texte sélectionné est traduit et envoyé.

C

Choisir la langue cible

Dans le popup, utilisez le menu déroulant "Traduire en" pour sélectionner parmi les 9 langues disponibles. La traduction se met à jour automatiquement.

D

App Desktop ou WhatsApp Web

Le toggle "Ouvrir via" dans le popup permet de choisir entre App WhatsApp (ouvre l'application installée sur votre PC) et WhatsApp Web (dans le navigateur). Ce réglage est mémorisé automatiquement.

Sur X.com, l'extension détecte automatiquement le texte du tweet. Inutile de sélectionner quoi que ce soit — ouvrez un tweet et cliquez sur l'icône.

Les liens présents dans le texte ne sont pas traduits. Ils conservent leur URL d'origine dans le message, quelle que soit la plateforme.

Sur LinkedIn, l'extension extrait automatiquement le texte du post affiché. Ouvrez un post et cliquez sur l'icône.

Langues disponibles

9 langues supportées

Français Anglais Espagnol Allemand Italien Portugais Arabe Chinois Japonais

La détection de la langue source est automatique. D'autres langues peuvent être ajoutées dans popup.html.

Maintenance

Mettre à jour l'extension

1

Remplacer les fichiers

Téléchargez la nouvelle version, décompressez et remplacez les fichiers dans le dossier existant sans changer son emplacement.

2

Recharger dans Chrome

Allez sur chrome://extensions, trouvez l'extension et cliquez sur l'icône de rechargement circulaire. Les changements sont appliqués immédiatement.
